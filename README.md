# Pythonボイラープレート

Python でツールを作る場合の最低限のコード

* .gitignore
* docker によるインフラのコード化・コンテナ化
* 設定ファイルの読み込み
  * settings.json
  * secrets.json
* コマンド引数のパース
* ロガー
  * ログの stderr 及びファイル出力
  * ログファイルのローテーション
* pytest
  * tests ディレクトリから src へのパス解決
* 起動シェル

など

参考： [Python 3 boilerplate - python-boilerplate.com](https://www.python-boilerplate.com/py3+executable)
